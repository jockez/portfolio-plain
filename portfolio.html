About

This project is still being actively developed. It's a PWA for my band, Exogenous, so we can keep track of time spent on band related stuff, hence the code is private. The project is mobile first because it's meant to be quick easy to report your time on the go. It works as good as mobile on both tablet and desktop.

The app features login, creating, editing and deleting sessions as well as top lists per month and total time reported. You can start a session and later come back to end it aswell. Sessions are updated in real time so the top lists are always accurate.

You can dig into the top lists and show the sessions reported per person if you want.
